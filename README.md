TestTask for Sequenia.

Небольшое пояснение к состояниям главного экрана. Когда сервер возвращает успешный ответ, фильмы заносятся в базу данных. Если во время запроса на получение фильмов с сервера в базе данных уже есть данные, они отображаются пользователю. Кроме того, если возникает ошибка, например, отсутствие интернета, данные из базы также показываются вместе с сообщением об ошибке.
